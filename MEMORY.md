## Brick AI（品牌长期记忆）
- 品牌名：**Brick AI**
- 名称含义：brick 代表 solid / reliable / foundational / timeless；AI 代表 intelligence / guidance / personalization。
- 一句话定义：**The intelligent foundation for home buying decisions.**

### 定位
- 品类：AI-powered property buyer’s agent（AI 买方代理）
- 市场空缺：传统房产平台偏 listings，缺少“帮买家做决策”的能力
- 目标用户痛点：选择过载、法律流程复杂、隐藏风险不透明、市场认知不足
- 定位陈述：通过智能推荐与引导式决策，帮助买家匹配偏好并规避隐藏问题

### 核心主张与口号
- Primary tagline：**Find the right home.**
- Product experience tagline：**Swipe homes. Learn your taste. Buy with confidence.**
- 品牌承诺：帮助用户从 confused browsing 走向 confident decisions
- 品牌愿景：让买房由“猜测”转向“智能引导”

### Brand Pillars（4）
1. Clarity：把复杂买房信息变清晰、可理解、可执行
2. Trust：透明洞察 + 数据支撑 + 清晰解释
3. Intelligence：通过互动学习偏好，AI 是思考伙伴而非筛选器
4. Empowerment：不替用户做决定，而是帮助做更好决定

### Voice & Tone
- 应有语气：intelligent / calm / clear / supportive / modern
- 避免语气：salesy、金融顾问腔、过度技术化 AI 术语
- 文案原则：说人话、强调清晰与信心、少 hype

### Core Message
- 核心理念：Buying a home should feel exciting, not overwhelming.
- 核心信息：帮助澳洲买家在偏好学习与智能建议中找到合适房源并做出有信心的决策
- 情绪价值：把不确定性转化为 clarity 与 confidence

### 体验原则
- Simple / Interactive / Insightful / Supportive

### 品牌人格
- 像一个：懂房产、冷静、可靠、现代、不过度推销的助手
